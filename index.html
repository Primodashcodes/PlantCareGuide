🔑 Paano Kumuha ng Gemini API Key

Ang Google Gemini API ay libre gamitin para sa personal at student projects. Sundin ang mga hakbang na ito:

1
Pumunta sa Google AI Studio
I-open ang iyong browser at pumunta sa aistudio.google.com — ito ang opisyal na developer platform ng Google para sa Gemini AI.
2
Mag-sign in gamit ang Google Account
I-click ang "Sign in" at gamitin ang iyong Gmail account. Hindi kailangan ng bagong account — ang existing Google/Gmail mo ay sapat na.
3
Tanggapin ang Terms of Service
Sa unang pagkakataon, magpapakita ng Terms of Service. Basahin at i-click ang "I agree" o "Accept" para magpatuloy. Awtomatiko rin itong gagawa ng Google Cloud project para sa iyo.
4
I-click ang "Get API Key"
Sa kaliwang sidebar o sa dashboard, hanapin at i-click ang "Get API key" button. Dadalhin ka nito sa API key management page.
5
Gumawa ng Bagong API Key
I-click ang "Create API key". Piliin ang "Create API key in new project" (inirerekomenda para sa mga baguhan). Hihintayin lang ng ilang segundo.
6
Kopyahin ang API Key
Kapag lumabas na ang key (nagsisimula sa AIza...), i-click ang copy button. Huwag ibahagi ang key na ito sa iba! I-paste ito sa Plant Identifier page ng app.
7
I-paste sa App
Pumunta sa 📷 Identify tab ng app, i-paste ang iyong API key sa input box na nagsasabing "AIza...", at handa ka na! Awtomatikong nate-save ito para hindi mo na kailangang i-paste ulit sa susunod.
💡
Tips:
  • Ang free tier ng Gemini API ay may daily quota — sapat para sa personal/student use.
  • Gamitin ang Gemini 1.5 Flash model — ito ang pinakamabilis at libre.
  • Huwag i-commit ang API key sa GitHub o ibahagi sa public.
  • Kung may error, subukan na i-refresh ang page at i-paste ulit ang key.
  • Ang key ay naka-save sa browser — hindi ito nawawala kahit i-refresh ang page.

🌿 About Plant Care Guide

Plant Care Guide is a web and mobile application designed to promote environmental awareness and support sustainable plant care practices. It addresses the growing lack of accessible information about plants and trees.

The application provides detailed care guides for 60 plant species, AI-powered plant identification using Google Gemini, GPS-based plant recommendations, and an intuitive mobile-friendly interface.

🌍
SDG 15: Life on Land Protecting terrestrial ecosystems through digital education

🚀 Emerging Technologies

🤖 AI (Google Gemini): Used for instant plant identification from photos. Simply take a picture and Gemini Vision identifies the plant and provides care information.

☁️ Cloud Computing: Plant data is delivered via a lightweight JSON database. Structured for easy updates and scalability to a full cloud backend.

📡 IoT (Mobile Sensors): Uses the device camera as an IoT sensor to capture plant images for AI analysis in real time.

🗺️ Geographic Technology: Uses the browser's Geolocation API (GPS) to detect the user's location and recommend plants suited to their climate zone.
